# Веб-приложение для учета свечей

## Описание

Данное веб-приложение разработано на языке Java.  
Приложение предназначено для работы с базой данных PostgreSQL и позволяет пользователю
регистрироваться, входить в систему и управлять списком свечей.



## Функциональность приложения

Веб-приложение поддерживает следующую функциональность:

- регистрация нового пользователя;
- вход пользователя в систему;
- добавление новой свечи;
- редактирование информации о свече;
- удаление свечи;
- поиск свечей среди добавленных пользователем.

## Инициализация базы данных

Начальная конфигурация базы данных выполняется автоматически при первом запуске Docker Compose.  
Для этого используется SQL-скрипт:
src/main/resources/init.sql

Скрипт создает необходимые таблицы и связи между ними.


## Запуск приложения через Docker


### Сборка проекта

В корневой директории проекта необходимо выполнить команду:
mvn clean package
После сборки будет создан файл web1.war в директории target.

Для запуска веб-приложения и базы данных необходимо выполнить команду:
docker compose up --build

В результате будут запущены два контейнера:
1. контейнер с веб-приложением (Apache Tomcat); 
2. контейнер с базой данных PostgreSQL.

После успешного запуска приложение будет доступно в браузере по адресу:
http://localhost:8080/web1/

Для проверки работы приложения необходимо:
1. зарегистрировать нового пользователя;
2. выполнить вход в систему;
3. добавить, отредактировать, удалить и найти свечи.
